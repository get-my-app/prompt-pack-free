Create a public prompt-pack repository for Free-tier prompts. The repository should contain Markdown prompts with YAML frontmatter, a README that explains safe public usage, and a clear rule that prompts must not contain private implementation details, credentials, customer data, or paid-roadmap internals.

Include prompts for creating and maintaining open-core projects, basic AI-agent workflows, public-safe documentation, and local-only tools. Keep the format simple enough to be copied into other prompt libraries later.
