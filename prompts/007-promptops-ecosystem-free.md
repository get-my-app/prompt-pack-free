Create the public Free tier of a PromptOps ecosystem repository. The repository should manage public-safe prompts, project specs, agent policies, run logs, subscription-tier notes, and generated task plans for multiple open-core projects.

Include a minimal structure for `shared/`, `projects/`, `prompt-packages/free/`, `meta/`, `tools/`, and docs. Keep the Free tier public-safe: no private implementation details, credentials, customer data, private roadmap internals, paid-tier automation, or private branch assumptions. Add a README, safety policy, prompt schema, and a small CLI or documented workflow for adding/listing prompts and recording run logs.
